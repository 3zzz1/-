总述
本平台采用 B/S 架构（推荐使用Vue3+Element Plus, SpringBoot+Mybatis Ruoyi框架），支持本地化部署、企业微信移动端嵌入，整合门户、驾驶舱、审计管理、作业、数据分析、协同、整改、档案、中介、知识、权限、流程引擎、AI 智能体全链路能力，覆盖审计计划、项目实施、数据分析、整改跟踪、档案归档、中介管理全生命周期，配套分级数据权限、自定义审批流、AI 智能辅助，实现高校审计业务数字化、标准化、可视化、智能化。
一、审计信息管理（基础数据底座模块）
审计信息管理是审计信息化建设底层基础底座，统一归集、维护全量审计静态业务数据，打通全系统数据调用通道，提升审计计划编制、资源调度、成果复用的科学性，支撑审计全覆盖落地，核心包含审计计划、审计对象、审计进度三大核心业务库。
（一）审计计划全生命周期管理
基础计划维护能力
1.支持按计划类型（经济责任审计 / 财务收支审计 / 专项审计 / 工程审计）、计划年度分批次批量新建年度审计计划，单批次独立归档管理；
2.附件上传：支持上传审计项目委托书、校党委会 / 校长办公会审批决议、上级审计机关下发任务文件、调研材料等附件，支持 PDF/Word/Excel/ 图片多格式在线预览；
3.审计方案绑定：按计划批次批量关联标准化审计方案库模板，支持单计划单独定制专属实施方案，方案随计划同步归档、可穿透查看完整内容。
项目批量编排与智能推荐
1.计划内项目支持手动逐条新增 + Excel 批量导入两种模式，支持批量修改项目类型、实施周期、审计组长、覆盖期间；
2.智能自动推荐待审对象：基于经责审计滚动周期、上一轮审计完成时间、干部任职年限、资金规模、基建项目竣工年限多维度规则，自动筛选近 3 年需启动审计的单位 / 领导干部，生成推荐清单，审计人员可一键批量加入年度计划；
3.筛选检索：支持按年度、项目类型、审计对象、实施状态、是否委托中介、计划审批状态多条件组合筛选，支持计划清单、项目清单导出 Excel 存档。
计划审批与动态调整
1.审批流绑定：可自主配置年度计划新增、计划调增 / 调减专属审批流程，全程留痕，实时查看每一级审批节点、审批人、审批意见；
2.计划预警跟踪：自动跟踪计划实际执行进度，若项目实施超计划预设期限，系统自动推送消息提醒审计组长、审计处负责人；
3.计划变更管理：年度计划正式审批通过后，支持调增新项目、调减存量项目、修改项目实施周期，每一次变更单独走变更审批流程，完整记录变更前后对比日志，变更记录永久留存。
计划穿透查询
年度计划汇总页面可穿透查看单项目完整信息、配套指导性文件、实施进度、底稿、报告、整改台账，实现 “计划 - 项目 - 成果” 一键联动。
（二）审计对象全维度建库管理
分为被审计单位库、经济责任领导干部库两大子库，统一维护、全局共享调用。
被审计单位库
1.分类分级管理：按校内职能部门、二级学院、直属后勤单位、校办企业、基建项目部、附属医院等维度分类建档；
2.完整档案字段：单位基础信息（编制、经费规模、分管校领导、财务联系人）、历年干部任免台账、历年审计记录、历次审计问题整改台账、专项资金项目清单；
3.自动全局调用：项目创建、通知书生成、底稿编制、整改下发全流程，自动回填单位名称、覆盖期间、联系人等基础信息，无需重复手动录入；
4.动态更新机制：支持手动更新、对接学校数据中台自动同步单位人事、经费变动信息，更新日志全程记录。
经济责任领导干部库
1.任职履历全记录：完整存储干部历任岗位、任职起止时间、分管业务、分管资金规模；对接学校人事数据中台，自动同步任免信息，无需重复录入；
2.审计履历归集：自动关联该干部所有已完成、在审经济责任审计项目，归集历次审计发现问题、整改完成情况、审计评价结论；
3.经责计划联动：智能匹配干部任职周期，自动生成滚动经责审计推荐清单，直接导入年度审计计划。
（三）审计进度可视化统筹管理
面向校领导、审计处管理层的全局调度视图，统筹校内自审、中介委托审计全部项目，保障审计全覆盖落地。
1.多维度可视化展示：以甘特图、柱状图、进度条展示年度全部已审批审计计划，按项目类型、实施阶段分组展示实施进度；
2.穿透式联动：点击项目名称直接跳转审计作业子系统，基于当前登录用户权限，展示项目实施方案、人员分工、底稿、疑点、报告、整改全流程信息；
3.资源调度支撑：直观展示各审计组长、中介机构人员负载，辅助管理层统筹调配审计人力，均衡项目排期；
4.进度预警：对超期未启动、实施滞后、报告逾期未提交的项目标红预警，自动推送消息至审计负责人。
二、审计作业管理（一线审计人员协同工作台）
审计作业管理是项目组核心协同操作模块，覆盖审计准备、实施、报告、归档全流程，标准化统一作业流程，抹平审计人员业务能力差异，降低审计风险，提升底稿、报告标准化程度，配套一站式审计工作台、多人在线协同、数据分析工具、多级复核机制。
（一）审计准备阶段
1.项目方案编制与分工 
o依据项目类型自动匹配审计方案库模板，一键生成初稿，支持表格、Word 两种编辑模式自由切换，审计人员可按需增删审计事项、调整核查程序；
o审计处长 / 审计组长按审计事项分配项目人员分工，明确每个成员负责核查范围、交付成果时限；
2.审计通知书线上下发 
o标准化模板自动填充审计对象、覆盖期间、项目组人员、需报送资料清单等库内基础信息，一键生成正式 Word 通知书；
o线上推送至被审计单位负责人、联络员一网通办待办，同步消息提醒，自动绑定该单位需提交的全套资料清单；
3.审前资料收集 
o被审计单位在线按清单分批上传资料，联络员提交后需单位负责人线上审核；
o审计人员支持上传线下纸质扫描件、外部调取资料，所有资料统一归集至项目资料目录，支持在线预览、检索。
（二）审计实施阶段（一站式审计工作台）
（1）多维度智能数据分析
内置完整大数据审计分析引擎，打通联网自动采集、离线导入的财务、业务标准数据库，提供向导式、拖拽式、自定义 SQL 三种分析工具，适配不同水平审计人员。
1.内置标准化分析模型：财务收支总额统计、分项支出用途分析、预算执行对比、采购业务全流程分析、各类收入归集核查、往来款项账龄分析、资金结算支付校验；
2.穿透查询能力：汇总分析图表可逐层下钻至明细凭证、合同、支付单据原始数据，支持按需抽样导出明细用于现场取证；
3.可视化展示：柱状图、折线图、饼图、热力图结合表格展示趋势、同比 / 环比对比、资金分布，异常数据自动标红风险预警；
4.疑点自动回流：分析发现的异常疑点一键同步至项目疑点台账，直接关联审计底稿作为取证佐证材料。
（2）标准化审计取证作业
1.取证单自动生成：录入审计问题后，一键生成 Word 格式取证单，支持在线 Word 协同编辑，被审单位联络员线上回复意见、上传佐证附件；
2.分类台账管理：按资金类、采购类、人事类、资产类对取证单、问题清单分组归档；
3.底稿自动生成：依据取证问题、审计事项模板自动标准化生成工作底稿，底稿内问题描述修改后，同步更新系统全局问题台账；
4.法规智能检索：编辑问题定性时，输入关键词自动检索法规制度库，法条一键插入底稿定性依据段；
5.全程操作留痕：取证、修改、附件上传、对方回复全部记录操作人、操作时间，满足审计质量督导要求。
（3）多级底稿复核流程
支持自定义 1~N 级复核流程（主审复核→组长复核→审计处负责人复核），复核人在线批注意见、标记修改点位，修改内容高亮比对，完整留存每一轮复核记录，复核未完成项目无法进入报告阶段。
（4）审计问题自动归集台账
所有经复核确认的审计问题自动归集至统一问题台账，自动绑定问题来源、定性法规依据、涉及金额、责任部门、责任干部，为后续整改跟踪子系统提供标准化数据源头。
（5）多人实时协同作业
同一项目内多名审计人员可同时在线编辑底稿、上传资料、录入疑点，系统自动锁定同一条目避免冲突，实时展示每个人任务完成进度，责任到人，项目组长全局监控实施进度。
（三）审计报告阶段
1.多版本报告线上编制 
o内审核稿、征求意见稿、正式定稿三套版本统一管理，依据项目类型自动匹配报告模板，内置金额汇总、问题统计自动计算公式，减少人工计算差错；
2.线上征求意见流程 
o征求意见稿一键推送至被审计单位负责人、经济责任人一网通办待办，双方在线回复意见、上传佐证材料，支持多轮意见交互，全程流程留痕；
3.修订比对功能：报告每一轮修改自动生成版本对比，增删内容高亮标注，大幅提升复核效率；
4.自定义内部审批流：报告初稿、终稿分别绑定预设审批流程，自动逐级推送至对应审批人线上签字批复。
（四）项目归档阶段
1.自动归集项目全生命周期资料：项目立项文件、通知书、实施方案、取证单、底稿、各版本报告、整改资料自动归集；
2.标准化档案目录：按照立项类、证明类、结项类、备查类、整改类五大分类自动生成档案目录，支持手动补充线下扫描资料；
3.归档审核机制：项目组长整理完成后提交档案管理员审核，审核通过后正式入库审计档案管理子系统；
4.权限化查阅：归档完成后，具备权限人员可检索预览目录，涉密档案需走借阅审批流程才可查看完整文件。
三、审计整改管理（审计成果落地监督模块）
统一管控校内审计、外部巡视、上级审计机关发现的全部问题，实现问题整改全生命周期线上闭环跟踪，自动预警逾期整改，多维统计整改成效，反向优化审计计划与审计重点。
（一）问题台账自动生成
1.内源问题：审计项目结项后，报告内全部问题自动同步至整改台账，携带问题金额、定性依据、责任单位、责任干部、整改时限；
2.外源问题：支持手动录入上级审计、巡视巡察、专项督查发现的外部问题，统一纳入整改跟踪体系；
3.台账全生命周期记录：完整留存问题下发、方案提交、整改填报、佐证上传、审核评价、销号全流程记录。
（二）整改任务下发与多渠道提醒
1.审计处批量下发整改任务，指定整改单位、联络人、整改截止时限，系统自动生成标准化整改通知书；
2.多端消息触达：系统站内消息、短信、企业微信移动端同步推送提醒，临近截止、超期未整改持续循环预警。
（三）整改单位端全流程操作
1.任务确认与分办：联络员接收任务后确认问题，拆分下发至具体整改责任人；
2.整改方案管理：责任人线上编制整改计划、整改方案，无法按期完成可提交延期申请；涉及制度机制短板的问题，支持申请转为长期持续整改，延期 / 持续整改均需线上审批生效；
3.整改材料填报：录入整改措施、整改成效，上传合同、凭证、制度文件等佐证附件，整改完成后发起销号申请；
4.自动生成整改报告：系统依据填报内容一键生成标准化整改报告，经单位负责人审批后提交审计处。
（四）整改审核与销号管控
1.审计处主审核对整改材料，区分整改完成、整改不到位两类结果；
2.审核通过：审批完成后问题正式销号，归档整改全套资料；
3.整改未达标：驳回销号申请，下发二次整改任务，明确补充整改要求；
4.整改流程全自定义：整改下发、延期、销号、持续整改全部流程支持自主配置审批节点，无需修改代码。
（五）整改数据多维分析
以表格、柱状图、饼图展示整改完成率、逾期率、问题类型分布、资金挽回规模、重复发生问题统计，自动识别高频风险领域，辅助审计处调整下一年度审计计划与审计重点；
（六）移动端协同
企业微信移动端支持整改消息接收、整改材料填报、审批、意见回复，随时随地处理整改业务。
四、审计依据库（审计标准化知识库底座）
统一归集审计工作全部参考依据，PC / 移动端双端随时检索，审计底稿、报告可一键引用法条、案例，分为政策法规库、问题风险库、审计案例库三大子库。
（一）政策法规库
1.分级分类存储：国家法律、部委规章、教育行业制度、校内财务 / 资产 / 采购 / 人事内部制度分层归档，分类清晰；
2.定期更新机制：支持管理员批量导入最新政策文件，过期法规标记失效，区分现行有效、废止文件；
3.多维度检索：支持全文模糊检索、按审计事项检索、按资金类型检索、按发文单位检索；
4.业务联动：编制底稿、定性审计问题时，关键词检索法条，一键插入至文档内，自动标注法规文号、条款原文；
5.法条关联维护：支持维护法规之间上位法、下位法、补充修订衍生关联关系，查看法条时同步展示关联文件。
（二）问题风险库
1.标准化风险条目：按财务、采购、资产、人事、基建分类维护常见审计风险点，每条风险绑定对应法规依据、典型表现形式；
2.快速检索：审计实施阶段可检索同类风险，提前锁定核查重点，辅助审计人员预判潜在漏洞；
3.自动沉淀：审计项目确认的新问题可一键入库更新风险库，持续丰富风险条目。
（三）审计案例库
1.自动入库：项目结项归档后，典型审计项目一键推送至案例库，支持管理员手动补充典型外部审计案例；
2.分类管理：按经责审计、财务收支、专项、工程审计分类，每条案例包含项目概况、发现问题、定性依据、整改措施、审计成效；
3.检索与学习：全文检索案例，支持在线预览完整案例材料，新审计人员可参考同类项目核查思路，统一审计作业标准，降低业务水平差异。
五、数据驾驶舱（审计决策可视化平台）
面向校领导、审计管理层，提供分层级、可穿透、实时更新的审计数据可视化大屏，细粒度数据权限隔离，B/S 架构支持本地化部署，无外网暴露风险。
（一）多维可视化穿透展示
1.多图表组件支撑：柱状图、折线图、饼图、热力图、甘特图、数据看板、明细表格自由组合；
2.层层下钻穿透：从全局总览→分类型统计→单项目汇总→底层凭证 / 底稿原始明细，一键直达源头数据；
3.实时 / 准实时数据更新：定时同步审计项目、整改、资金分析数据，支持自定义刷新周期，保障数据时效性；
4.自定义页面布局：各层级用户可自主拖拽组件调整驾驶舱看板布局，保存专属个人视图模板。
（二）分级细粒度数据权限隔离
1.数据分级管控：校领导、审计处长、项目组长、被审计单位负责人、中介人员分配不同数据查看范围；
2.差异化界面展示：登录账号自动匹配权限，仅展示自身授权范围内数据： 
o校领导：全校审计总览、全部项目进度、整体整改成效、大额资金风险预警；
o审计处长：全量审计数据、人员负载、中介考核、问题分类统计；
o项目组长：仅本人负责项目进度、疑点、整改跟踪；
o被审计单位：仅本单位审计与整改数据；
3.财务数据强隔离：未授权用户完全无法查看任何财务明细、凭证数据，从底层拦截越权访问。
六、AI 智能体嵌入功能（全流程智能辅助）
平台内置专属审计 AI 智能体，打通法规库、项目库、数据分析引擎，支持多轮对话问答，嵌入门户、作业、移动端全场景。
1.AI 政策法规问答 
o自然语言提问审计相关法规、校内制度，AI 自动检索法规库，精准匹配对应法条，给出解读、适用审计场景；
o支持长文本底稿、报告上传，AI 自动匹配定性依据，提示法条引用遗漏。
2.AI 审计项目信息查询 
o自然语言查询项目进度、问题清单、整改完成情况、历年审计记录；
o例如：“XX 学院 2024 年经责审计整改完成情况”，AI 直接汇总数据并生成简明报表。
3.AI 智能数据分析 
o自然语言下发分析指令，如 “统计本年度采购超 50 万无招标项目”，AI 自动调用数据引擎执行分析，生成可视化图表、疑点清单；
o自动识别数据异常、资金波动，生成风险提示，一键回流至审计项目疑点台账。
七、自主配置审批流程引擎
内置可视化流程设计器，审计处管理员可视化拖拽配置全类型业务审批流，支持线上电子审批，无需开发人员修改代码。
1. 支持配置的全部审批场景
年度审计计划新增 / 调增减审批、审计项目立项审批、审计方案审批、审计报告多轮审核、征求意见稿审批、整改延期 / 持续整改审批、问题销号审批、档案借阅审批、数据资源授权审批、中介机构准入考核审批。
2. 流程自定义能力
1.可视化拖拽节点：新增审批人、会签节点、条件分支、抄送节点；
2.分支条件配置：可按项目类型、资金规模、整改金额自动分流至对应审批层级；
3.审批人灵活绑定：固定岗位、指定人员、角色分组三种模式任选；
4.流程模板保存：配置完成的审批流程保存为模板，新建同类业务直接复用。
3. 线上审批配套能力
1.自动分发待办：业务发起后，自动推送至对应审批人门户、移动端待办；
2.审批意见留痕：支持文字批注、附件上传、驳回退回并标注修改要求；
3.移动端审批：企业微信手机端可查看审批材料、填写意见、同意 / 驳回；
4.流程追溯：每一条审批单完整记录流转节点、操作人、操作时间、意见，永久存档可检索
八、分级细粒度权限管理体系（RUOYI框架已有类似功能，不作为单独开发模块！）
采用机构 + 用户 + 角色 + 数据 + 功能五维权限模型，区分四类核心用户群体，权限可灵活勾选配置，无需代码修改。
1.校领导权限
功能权限：审计综合门户、审计驾驶舱、全部审计报告阅览、全校整改统计、档案借阅审批；
数据权限：全校所有审计汇总数据，屏蔽底层明细财务数据，如需查看明细需单独申请临时授权。
2.被审计单位负责人 / 联络员权限 
o单位负责人：仅查看本单位所有审计报告、本单位问题整改台账、整改复核结果；
o联络员：新增权限，在线提交审计资料、回复取证单、填报整改材料，仅可操作本单位业务数据，无法查看其他单位信息。
3.审计部门全员权限 
o普通审计人员：查看分配给自己的项目资料、开展作业、数据分析、检索依据库；
o项目组长 / 主审：增加项目维护、人员分工、底稿复核、报告编制、整改评价权限；
o审计处长：全系统全部功能、全量数据访问权限，流程审批、中介考核、系统配置权限。
4.中介社会审计机构人员权限
仅在审计项目服务周期内开通临时授权，仅可访问分配的对应项目数据、资料、底稿，项目结项后权限自动回收；仅具备作业录入权限，无全局统计、驾驶舱、其他项目访问权限。
权限底层机制
1.功能权限：控制菜单、按钮、编辑 / 删除操作可见性；
2.数据权限：按单位、项目、数据类型隔离，越权数据自动隐藏；
时效权限：中介、临时借阅权限支持设置到期自动回收，全程记录授权日志。
